Detailed Description

A basic Quaternion class.

GPU:
Quaternion class with components \( x, y, z, w \) such that \( q = x\textit{i} + y\textit{j} + z\textit{k} + w\). Basic operators provided are \(\times\): multiplication and \(+\): addition. \(^{-1}\): inverse is provided through quaternion::inv() method. Can be used in both host and device code.

CPU:
Quaternion class with components \( x, y, z, w \) such that \( q = x\textit{i} + y\textit{j} + z\textit{k} + w\). Basic operators provided are \(\times\): multiplication and \(+\): addition. \(^{-1}\): inverse is provided through quaternion::inv() method.
